Mechanics Conversions

Mass, Force, and Pressure conversions follow the same principle: a base unit and conversion factors are used to apply the general formula:

Final Value = (Initial Value Ã— Source Factor) / Target Factor

  • Mass: The base unit is the Kilogram (kg).
  • Force: The base unit is the Newton (N).
  • Pressure: The base unit is the Pascal (Pa).

For any conversion, the value is first transformed to its equivalent in the base unit and then to the target unit.
